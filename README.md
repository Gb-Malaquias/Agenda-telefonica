# Agenda Telefônica – C#

Aplicação de console em C# para cadastro e gerenciamento de contatos.

## Funcionalidades
- Adicionar contatos
- Listar contatos cadastrados
- Buscar informações
- Remover contatos

## Objetivo do Projeto
Projeto voltado à prática de:
- Estruturas de dados
- Lógica de programação
- Organização de informações em memória

## Tecnologias Utilizadas
- C#
- .NET
- Git
